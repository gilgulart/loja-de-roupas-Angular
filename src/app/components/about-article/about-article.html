Estrutura do Projeto

Antes de iniciar qualquer linha de código, traçamos um direcionamento claro para o desenvolvimento do projeto. Definimos que ele seria um e-commerce voltado para uma marca de roupas Streetwear, o que nos levou naturalmente a adotar a abordagem Mobile First, considerando que a maior parte do tráfego provavelmente viria de dispositivos móveis.

A construção da identidade visual foi uma das etapas mais importantes do processo. Buscamos unir o visual urbano e despojado do Streetwear com referências psicodélicas inspiradas nos anos 70, criando uma estética moderna e autêntica. O ChatGPT foi utilizado como ferramenta de apoio na geração de imagens e também na elaboração de um Style Guide para o projeto.

Acreditamos que, para um site moderno, responsivo e visualmente consistente, é essencial estabelecer padrões bem definidos para cores, tipografia e componentes. Nesse sentido, o Style Guide teve um papel fundamental no desenvolvimento, servindo como base para manter a coerência visual e facilitar a organização da interface.

Também mantivemos uma estrutura de código modular e organizada. Iniciamos a estilização pelo arquivo index.css, responsável por importar os demais arquivos CSS utilizados nas páginas HTML. Primeiro aplicamos os estilos globais do projeto, definindo variáveis no :root para cores e tipografia, e somente depois partimos para a estilização individual dos elementos da interface.

Além disso, adicionamos duas animações simples para enriquecer a experiência visual do usuário: um efeito de marquee no header, exibindo mensagens em movimento, e uma animação sutil no background da aplicação.

A colaboração da equipe aconteceu através do GitHub. Tornamos o projeto open source para que todos os colaboradores pudessem realizar forks e trabalhar em suas próprias versões do repositório, facilitando a integração e o desenvolvimento coletivo.
